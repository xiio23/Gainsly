# 🪙 Gainsly

**The most minimal and elegant crypto profit calculator for modern traders.** *Kalkulator profit crypto minimalis yang membantu Anda menghitung potensi keuntungan dengan konversi otomatis ke Rupiah.*

---

## ✨ Features / Fitur Utama

* **📊 Minimalist Profit Calculator** / *Kalkulator Profit Minimalis*
    Calculate ROI and net profit easily with a clean and premium interface.
    *Hitung ROI dan profit bersih tanpa ribet dengan antarmuka yang bersih dan premium.*

* **🇮🇩 Automatic Rupiah Conversion** / *Konversi Otomatis Rupiah*
    Forget manual math, all results are instantly converted to the latest IDR rate.
    *Lupakan kalkulator manual, semua hasil investasi langsung dikonversi ke kurs IDR terkini.*

* **⚡ Real-time Prices** / *Harga Real-time*
    Accurate data from exchanges to ensure your profit analysis remains precise.
    *Data akurat dari exchange untuk memastikan perhitungan potensi cuan Anda tetap presisi.*

* **📱 24/7 Access** / *Akses 24/7*
    Analyze your market strategy and calculate your gains anytime, anywhere.
    *Analisis pasar dan hitung strategi investasi Anda kapan saja, di mana saja.*

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **Deployment:** Vercel

## 🚀 Getting Started / Cara Menjalankan

1.  **Clone repository:**
    `git clone https://github.com/xiio23/Gainsly.git`

2.  **Install dependencies:**
    `npm install`

3.  **Run development server:**
    `npm run dev`
